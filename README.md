# Suno AI Knowledgebase

Welcome to the **Suno AI Knowledgebase**.

This repository is the starting point for a fully functioning GitBook-style documentation site for Suno AI. It is organized to support both beginners and advanced creators with structured guides, reference material, and version-aware documentation.

## What this knowledgebase covers

- Suno model evolution from V2 through V5.5
- Suno Studio workflows and editing systems
- Prompt engineering frameworks
- Lyrics, structure, and AI songwriting guidance
- Rights, ownership, and commercial-use basics
- Troubleshooting, glossary, and release tracking

## Recommended reading path

### New users
1. Start here
2. Read `introduction/quick-start.md`
3. Review `core-documentation/suno-at-a-glance.md`
4. Explore `creation-workflows/creation-modes-overview.md`
5. Use `prompt-engineering/handbook.md`

### Advanced users
1. Review `prompt-engineering/handbook.md`
2. Explore `core-documentation/suno-studio-master-guide/overview.md`
3. Compare model pages in `core-documentation/suno-model-encyclopedia/`
4. Use `troubleshooting-reference/troubleshooting-and-output-repair.md`

## Structure

- `introduction/` — onboarding, quick start, standards
- `core-documentation/` — platform overview, models, studio
- `creation-workflows/` — generation and editing workflows
- `prompt-engineering/` — prompting systems and formulas
- `songwriting/` — lyrics, structure, genre, vocal direction
- `rights-and-business/` — ownership, monetization, legal basics
- `troubleshooting-reference/` — fixes, glossary, comparisons, release notes

## Status

This is an initial GitBook-ready foundation created from the master guide and intended for continued expansion.
