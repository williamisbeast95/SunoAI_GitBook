# Prompt Engineering Overview

Prompt engineering is the discipline of shaping inputs so Suno produces stronger, more intentional results.

## Why It Matters

Better prompts improve:

- genre accuracy
- vocal direction
- emotional tone
- arrangement feel
- consistency across generations

## Core Principle

A weak prompt asks for a song. A strong prompt defines the song's identity.

## What This Section Covers

This section explains how to structure prompts, guide style, shape vocals, and refine outputs through iteration.
