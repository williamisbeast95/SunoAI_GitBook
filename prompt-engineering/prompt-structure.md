# Prompt Structure

## Overview
A strong prompt has clear structure and intent. Instead of vague ideas, it defines key attributes of the track.

## Core Elements

- genre
- mood
- tempo or pacing
- vocal style
- instrumentation

## Example Structure

Genre + Mood + Vocal Style + Instrumentation + Energy

## Tips

- be specific but not overloaded
- avoid contradictions
- keep structure consistent across iterations

## Goal

Create prompts that produce repeatable, predictable results.
