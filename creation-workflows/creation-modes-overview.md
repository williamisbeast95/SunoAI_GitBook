# Creation Modes Overview

Suno supports multiple creation paths depending on whether you want speed, control, revision, or transformation.

## Main Modes

- Simple mode for fast generation
- Custom mode for more intentional prompting
- Extend for building from an existing result
- Remix for reinterpreting an existing result
- Remaster for improving polish

## How to Choose

Use simple mode when exploring ideas quickly. Use custom mode when you already know the sound you want. Use post-generation tools when a draft has promise but needs work.
