# Iteration Workflows

## Overview
Strong results usually come from iteration rather than a single generation. Studio works best when you treat a track as something to refine in passes.

## Common Iteration Paths

- generate then extend
- generate then remix
- generate then remaster
- edit in Studio between revisions

## Practical Workflow

1. Generate an initial draft
2. Identify the strongest section
3. Preserve what works
4. Fix weak areas through extension, remixing, or cleanup
5. Repeat until the track feels intentional

## Best Practices

- change one major variable at a time
- keep notes on what improved quality
- compare versions before replacing a draft
