# Editing Basics in Suno Studio

## Overview
Editing in Suno Studio allows you to refine generated tracks by adjusting structure, timing, and flow.

## Core Actions

- Trim unwanted sections
- Rearrange segments
- Adjust transitions
- Test alternate variations

## Best Practices

- Keep strong sections, remove weak ones
- Focus on flow between parts
- Iterate instead of expecting perfection in one pass

## Tips

Work in small changes. Large edits can break coherence.
