# How to Use This Knowledgebase

This knowledgebase is designed to work both as a learning path and as a reference system.

## If you are new

Start with the introduction pages, then move into the platform overview and creation workflow guides.

Recommended path:

1. Welcome
2. Quick Start
3. Suno at a Glance
4. Creation Modes Overview
5. Prompt Engineering Handbook

## If you are experienced

Use the GitBook as a reference library.

Recommended path:

1. Model Encyclopedia
2. Studio Master Guide
3. Best Practices by Model Version
4. Troubleshooting and Output Repair

## How pages are structured

Most pages follow a simple pattern:

- overview
- best uses
- limitations
- practical tips
- related pages

## How to get the most value

Use overview pages for orientation and deeper pages for applied workflows. Return to troubleshooting when outputs drift or quality drops.
