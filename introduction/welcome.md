# Welcome

Welcome to the Suno AI Knowledgebase.

This GitBook is designed to be a structured documentation system for Suno AI, its models, Studio environment, workflows, and creator strategies.

## Who this is for

- New creators learning Suno
- Advanced users refining outputs
- Prompt engineers building repeatable systems
- Writers and producers exploring AI-assisted music creation

## What you will find here

This space is organized into onboarding, core documentation, workflow guides, prompt engineering references, songwriting systems, rights guidance, and troubleshooting.

## How to use this GitBook

Start with the introduction pages if you are new. If you already understand the platform, jump to the model encyclopedia, Studio guides, or prompt engineering section.
